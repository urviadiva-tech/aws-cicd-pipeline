## 🚀 CI/CD Pipeline (AWS)

###  Objective
Automate deployment of a static website using AWS services.

---

### ⚙️ Services Used
- Amazon S3 (Static hosting)
- AWS CodePipeline
- GitHub (Source)

---

### 🔄 Pipeline Steps

1. Created S3 bucket for hosting website  
2. Enabled static website hosting  
3. Uploaded HTML files  

4. Connected GitHub repository  
5. Created AWS CodePipeline  
6. Selected GitHub as source  

7. Configured deployment stage (S3)  
8. Pipeline triggered automatically on code push  

---

### Screenshots Included
- S3 bucket setup  
- CodePipeline configuration  
- GitHub integration  
- Deployment success  

---

###  Output
Website successfully deployed and updated automatically via CI/CD pipeline.
